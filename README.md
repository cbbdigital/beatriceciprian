Site de nuntă cu invitații personalizate, RSVP și dashboard pentru nuntași.

## Tehnologii
- React + Vite
- Supabase (baza de date)
- Netlify (hosting recomandat)

## Rulare locală

```bash
npm install
npm run dev
```

## Deploy pe Netlify

1. Push pe GitHub
2. Mergi pe [netlify.com](https://netlify.com) → **Add new site** → **Import from GitHub**
3. Build command: `npm run build`
4. Publish directory: `dist`
5. Click **Deploy**

## Linkuri invitați

Fiecare invitat primește un link unic:
```
https://site-tau.netlify.app/?invite=nume-prenume
```

Invitații se adaugă din **Dashboard** (`?invite=admin`).
